It is very easy to install and use the wordpress paypal button plugin script g with the best options and configurations that are not available in any other plugin. Even A Beginner or a kid can insert paypal payment buttons in wordpress pages and posts! Grab it and begin selling any stuff from your Word-Press site in about 10 minutes from now!

WordPress is the easiest and most powerful CMS to build any site on any niche and you can make it the most powerful sales platform or shopping website to sell your stuff. This is possible with WP plugin like "Paypal Button Creator" !

Just imagine being able to use your mouse to make a few clicks and setup a shopping cart or a simple buy now paypal button to receive payments via paypal and deliver the product file instantly to your customer after purchase - all with your WordPress website. It really is that simple.

* Tested, and working, in Wordpress 3.0
 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload all files in the package to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Manage the Options and Configuration from WP Admin panel.

== Frequently Asked Questions ==

= Which Wordpress Versions is it compatible =

It works from 2.5 to 3.0 (latest).

= Can we Add Your Own button graphics =

Yes, you just have to add button graphics in lib/img folder.
